# Telegram Oxford 3000 Bot

ربات تلگرام برای یادگیری ۳۰۰۰ لغت انگلیسی ویژه کاربران فارسی‌زبان.

## راه‌اندازی

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
copy .env.example .env
python main.py
```

داخل `.env` مقدار `BOT_TOKEN` را با توکن واقعی BotFather جایگزین کن.

برای ادمین شدن، در تلگرام دستور `/myid` را به ربات بفرست و عدد برگشتی را در `.env` بگذار:

```env
ADMIN_USER_IDS=123456789
```

اگر اتصال مستقیم به تلگرام نداری، پروکسی محلی را هم تنظیم کن:

```env
TELEGRAM_PROXY_URL=http://127.0.0.1:10809
```

برای تست داده‌ها:

```powershell
python main.py --check
```

## قابلیت‌ها

- تمرین سه‌گزینه‌ای با یک کلمه انگلیسی و سه معنی فارسی
- ورود خودکار کلمه به لیست آموخته‌شده‌ها بعد از ۷ پاسخ درست پشت سر هم
- ریست شدن streak همان کلمه بعد از جواب اشتباه
- دکمه «این کلمه را می‌دانم»
- دکمه «دیگر این کلمه را نشان نده»
- نمایش نمونه جمله برای هر کلمه
- مرور گاه‌به‌گاه کلمات آموخته‌شده
- برگشت کلمه آموخته‌شده به یادگیری بعد از جواب اشتباه
- لیست آموخته‌شده‌ها، آموخته‌نشده‌ها و کلمات حذف‌شده
- تنظیمات و بازنشانی کامل
- آزمون تعیین سطح اختیاری در شروع و از صفحه اصلی با حدس معنی کلمات
- آمار کلمات آموخته‌شده امروز و کل
- پنل `/admin` برای دیدن آمار کلی ربات

## نکته داده‌ای

داده‌ها از `F:\Apps\ReactNative\en_words\src\data\dictionary.ts` گرفته شده‌اند. از ۳۰۰۰ ورودی اول استفاده شده و سطح‌بندی بر اساس ترتیب فایل انجام شده است: ۹۰۰ کلمه A1، ۸۰۰ کلمه A2، ۷۰۰ کلمه B1 و ۶۰۰ کلمه B2.
